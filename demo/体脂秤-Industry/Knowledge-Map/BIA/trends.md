# BIA 技术演进趋势

BIA技术演进路径：单频 → 双频 → 多频/DF-BIA。单频BIA (50kHz) 是当前消费级市场的主流方案，成本最低但精度受限。

双频BIA (50kHz+250kHz) 能够更好地分离细胞内液与细胞外液，提升体脂估算精度。DF-BIA (分布式频率BIA) 是当前前沿，如Renpho MorphoScan和Withings Body Scan，通过扫描多个频率点获取阻抗谱，进行更全面的身体成分分析。

下一代技术方向是AI个性化校准：利用个体纵向数据（而非群体平均值）进行个性化模型训练，从而大幅提升单次测量的准确性和一致性。这需要用户长期使用同一设备积累数据，也意味着品牌粘性的提升。
