## 使用建议

### 如何用 Obsidian 打开

1. 打开 Obsidian → Open folder as vault → 选择 `体脂秤-Industry` 文件夹
2. 使用 Graph View 查看知识地图的节点连接
3. 使用 Backlinks 功能追踪概念间的引用关系
